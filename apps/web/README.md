# Secure Transactions Web App

Next.js frontend for encrypted transaction management.

## Quick Start

```bash
pnpm install
pnpm dev
```

Visit http://localhost:3000

## Configuration

`.env.local`:
```
NEXT_PUBLIC_API_URL=http://localhost:3002
```
